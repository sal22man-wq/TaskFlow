اللغة الأساسية:
العربية (واجهة كاملة باللغة العربية)

🎯 الهدف الأساسي

منصة شاملة لإدارة المهام والتعاون الجماعي مصممة خصيصاً للفرق العربية، مع تكامل مع تطبيق الواتساب لضمان التواصل الفعال وقياس رضا العملاء.

🚀 الميزات الأساسية

📋 نظام إدارة المهام المتقدم
  • إنشاء وتتبع المهام مع تفاصيل شاملة
  • نظام الترقيم التلقائي للمهام
  • حالات متعددة: قيد التنفيذ، مكتمل، متأخر، ملغي
  • مستويات أولوية: عالية، متوسطة، منخفضة
  • تتبع التقدم مع شريط تقدم مرئي
  • تواريخ استحقاق مع تنبيهات
  • نظام تعليقات داخلي لكل مهمة
🔐 نظام الصلاحيات الثلاثي
  • مدير النظام: إدارة كاملة للنظام والمستخدمين
  • المشرف: إنشاء المهام وإدارة الفريق
  • المستخدم العادي: عرض المهام المسندة فقط
  • تحكم دقيق في الصلاحيات
  • تغيير الأدوار بسهولة
📱 التكامل مع الواتساب
  • إرسال تلقائي لتقييم رضا العملاء
  • ثلاثة مستويات تقييم: غاضب، راضي، راضي جداً
  • تتبع الردود وتحليل النتائج
  • مسح QR Code للربط السريع
  • إحصائيات شاملة للتقييمات
🏆 نظام النقاط والمكافآت
  • نقاط تلقائية عند إكمال المهام
  • ترتيب الأعضاء وقائمة أفضل المؤدين
  • إحصائيات الفريق ومتوسط النقاط
  • نظام حوافز لتشجيع الأداء المتميز

🖥️ التكنولوجيا والمواصفات التقنية

⚛️ تكنولوجيا الواجهة الأمامية

  • React 18: إطار عمل حديث للواجهات
  • TypeScript: لضمان جودة الكود
  • Tailwind CSS: تصميم متجاوب وحديث
  • Radix UI: مكونات واجهة متقدمة

🖥️ تكنولوجيا الخادم الخلفي

  • Node.js: بيئة تشغيل سريعة
  • Express.js: إطار عمل خادم قوي
  • PostgreSQL: قاعدة بيانات متقدمة
  • Drizzle ORM: تفاعل آمن مع قاعدة البيانات

🛠️ المكتبات المساعدة

  • WhatsApp Web.js: تكامل مع الواتساب
  • React Query: إدارة حالة الخادم
  • Zod: التحقق من صحة البيانات
  • Date-fns: معالجة التواريخ
